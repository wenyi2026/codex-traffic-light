# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.2.x | Yes |

## Reporting a Vulnerability

Please open a GitHub issue with a clear description of the problem, reproduction steps, and affected version.

Do not include private Codex conversation contents, tokens, credentials, or other sensitive data in public reports.

## Local-Only Design

Codex Traffic Light is designed as a local desktop utility. It does not require a server and does not upload Codex session data.
