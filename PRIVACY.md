# Privacy

Codex Traffic Light is a local Windows desktop utility.

## What It Reads

The app may read local Codex session/status files and process activity to infer whether Codex is idle, running, or waiting for attention.

Typical local paths include:

```text
%USERPROFILE%\.codex\sessions
%USERPROFILE%\.codex\archived_sessions
%LOCALAPPDATA%\CodexTrafficLight
```

## What It Sends

The app does not send your Codex conversations, session files, status files, or local process data to any server.

## Notes

If you publish screenshots, logs, or issue reports, review them first and remove private task details.
