# Changelog

## v1.2.0

- Added polished dark and light UI modes.
- Added theme persistence across restarts.
- Added close and minimize controls.
- Added manually resizable floating widget.
- Added Chinese tray menu.
- Added automatic launch with Codex through a lightweight watcher.
- Added automatic window close when Codex fully exits.
- Improved status detection rules:
  - Green: Codex is idle.
  - Yellow: Codex is thinking, responding, running, or using tools.
  - Red: approval, error, blocked state, or human intervention is required.
- Added installer package and user manual.

## v1.0.0

- Initial desktop traffic-light widget prototype.
